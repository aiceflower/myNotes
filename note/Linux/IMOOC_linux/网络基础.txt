osi的七层模型：
	应用层(APDU)、表示层(PPDU)、会话层(SPDU)、传输层(TPDU)、网络层(报文)、数据链路层(帧)、物理层(比特)。
	上三层给用户服务的，下四层数据传递。
	物理层：设备之间的比特流的传输、物理接口、电气特性。
	数据链路层：用MAC地址访问媒介
	网络层：提供逻辑地址、选择路由
	传输层：可靠与不可靠的传输、传输前的错误检测、流控。
	会话层：对应用会话的管理、同步
	表示层：数据的表现形式，特定功能的实现如 加密
	应用层：用户接口
TCP/IP四层模型：
	应用层、传输层、网际互联层、网络接口层。
	
网络查看命令：
	ifconfig 查看当前生效的ip地址
	ifdown 网卡设备名  --禁用网卡
	ifup 网卡设备名    --启动网卡
	netstat 选项
	-t:列出TCP协议端口
	-u:列出UDP协议端口
	-n:不使用域名与服务名，而使用ip地址和端口号
	-l:仅列出在监听状态网络服务
	-a:列出所有的网络连接
	应用：netstat -tuln --查看当前计算机开启了哪些端口
		netstat -an | grep ESTABLISHED | wc -l  --查看当前计算有多少人连接
		netstat -rn --查看网关 用 route -n也可以查看，最后一行
	查看dns
		nslookup ip地址
		或 nslookup 回车 再输入 server
网络测试命令：
	ping ip地址  --设置次数  -c 3 
	telnet ip地址 端口 
	traceroute ip地址  --路由查看 -n 选项使用ip不使用域名，速度更快
	wget 网络地址  --下载命令

ssh命令
	远程管理指定linux服务
	ssh 用户名@ip地址
scp [-r] 用户名@ip地址：文件路径 本地路径   --下载文件
scp [-r] 本地文件 用户名@ip地址：上传路径   --上传文件 -- -r 传送目录 
scp用于linux与linux之间传送文件
