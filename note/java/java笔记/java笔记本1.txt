作业：11  14   15

1.this只能在类中使用，而不能在外部使用。this只能在方法里使用不能在修饰为static的方法里使用。

2.静态的方法只能访问静态的变量；
  非静态的方法既可以访问静态的变量也可以访问非静态变量。

3.如果“仅仅是”返回类型、修饰符不一样不能够构成重载。

4.方法覆盖：子类的方法返回类型，参数，方法名称要和父类
完全一样、子类方法不能不能缩小父类方法的访问权限，增大可以。

5.多态通过继承和接口实现构成多态。

6.多态：一个引用在不同情况下的多种状态。

7.当一个类继承的父类是抽象类的话，需要我们把抽象类中的所有
  抽象方法全部实现。（方法的重载）

8.用abstract 修饰的类（方法）就叫做抽象类（方法）。
  a) 抽象类不能被实例化。
  b)类如果包含了抽象方法，那么这个类必须声明抽象类。
  c)抽象方法不能有主体。“｛｝”

9.当一个类实现了一个接口，就要求该类把这个接口的所有方法都实现。

10.接口不能被实例化、接口中的所有方法都不能有主体、一个类可以
   实现多个接口、接口中可以有变量但不能是private protected修饰
(只能是public final static),接口的变量本质是static的而且是final.
无论你是否加static. 常常反常用的变量定义在接口中作为全局变量使用，
（访问：接口名.变量名）、接口不能继承类但是可以继承其它接口。

11.接口是更加抽象的抽象类，抽象类里的方法可以有方法体，接口里的
所有方法都没有方法体。接口体现了程序设计的多态和高内聚低偶合的设计思想。

12.java继承是单继承。

13.方法不想被覆盖，变量不想被修改，或类不想被继承就用final.  
   如果是final要在定义的时候就附初值，且以后不能修改。

14.数组可以存放多个 同一类型的数据。

15。“==”比较int float double.比较字符串内容是否相等时用equal

16。局部变量描述的是这个方法体内的属性，而成员变量描述的是这个对象里的属性。
成员变量（堆里创建有默认值）可以被 public  private protected default static final修饰
而局部变量（栈里创建无默认值）只能被final修饰。

17。import语句要在package语句之后使用。

18。类不能多继承，但可以通过接口来实现多层继承。

19。接口里定义的变量为常量。

20.在try  catch finally语句中 finally 是必须被执行的。catch和finally必至少
有一个存在且它们中间不能有其它除了注释以外的其它语句。可以存在多个catch但
其中一个执行后，后面的将不再执行。

21。如果程序中未出现捕获异常，但程序中仍对该语句进行捕获异常处理，程序
运行时会出错。
