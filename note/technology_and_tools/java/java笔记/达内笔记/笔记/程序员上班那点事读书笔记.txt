优胜劣汰

知己知彼,百战不殆
知己:要对自身的情况了解,要找准自己的定位。
知彼:要对未来工作岗位的一切做到尽可能的清楚

修炼第一层境界:剑指四方,试问天下谁与争锋
修炼第二层境界:世界如此之大,要学的东西很多
修炼第三层境界:最深即最浅,最浅即最深,不过如此

"古今之成大事业、大学问者,必经
过三种之境界。'昨夜西风凋碧树,独上高楼,望尽天涯路',此第一境也。'衣带渐宽终不悔,为伊消得人憔悴',此第二境也。'众里寻他千百度,蓦然回首,那人却在,灯火阑珊处',此第三境也。"

程序员的发展路径,就是程序员人生路径,从最初的"看山是山,看水是水",到后来见的多了悟到"看山非山,看水非水",没想到最终发现"看山还是山,看水还是水"。

任何事业,均需如下才可成功:第一要立志,第二要思考,第三要奋斗。

又有人说人生境界如陶渊明《桃花源记》中所写的"初极狭,才通人。复行数十步,豁然开朗。"

工作经验是:具备职位所需技术的精度和深度,最好是对那个技术非常精熟,具有为公司服务的意识,有为公司创造价值的能力,至少有为公司节约成本的能力。

员,看到他以后愣是想不起来他的名字,再比如,一个很常见的字,提起笔来就是忘了怎么
