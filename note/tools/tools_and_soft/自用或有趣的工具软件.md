### 自用或有趣的工具软件

### 一、windows

1. typora 

   编写MarkDown文件 

2. QuickLook

   windows空格预览文件工具

3. Listary

#### 二、Linux

1. zsh

   linux 中好用的shell软件  

```shell
yum instaal zsh #安装
cat /etc/shells #查看
chsh -s /bin/zsh #切换
```

2. 一键安装vim

```shell
wget https://raw.github.com/ma6174/vim/master/setup.sh
sh setup.sh
```
